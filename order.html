Flexbox Layout

Сайт создан в целях демонстрации модуля Flexbox Layout и обучения ему

order

Это свойство применяется к flex-элементу и управляет порядком внутри flex-контейнера.

order: 0;

По умолчанию. Все элементы следуют в порядке расположения в HTML коде.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 0;

flex-shrink: 1;

order: 0;

}
