align-self

Это свойство позволяет переопределить выравнивание по умолчанию (или заданное с помощью свойства align-items) для отдельных flex-элементов. Применяются такие же значения, как у align-items.

align-self: stretch;

Установлен по умолчанию. Flex-элемент будет растягиваться на всю длину поперечной оси, но соблюдает размеры установленные элементу.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex: initial;

}

.flexElement:nth-of-type(2) {

flex: initial;

align-self: stretch;

}

align-self: baseline;

Flex-элемент выравнивается по базовой линии. Если элемент со значением baseline один, он выстраивается по своей базовой линии.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex: initial;

}

.flexElement:nth-of-type(odd) {

flex: initial;

align-self: baseline;

}
