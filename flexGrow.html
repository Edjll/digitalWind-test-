flex-grow

Это свойство применяется к flex-элементу и определяет возможность элемента увеличиваться, при необходимости. Оно принимает безразмерное значение в качестве пропорции, которое определяет какое количество свободного пространства основной оси внутри контейнера должен занимать элемент. Нельзя указывать отрицательные значения.

flex-grow: 0;

По умолчанию. Flex-элементы не растягиваются.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 0;

}

flex-grow: 1;

Свободное пространство основной оси flex-контейнера будет равномерно распределено между всеми flex-элементами.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 1;

}

1
2
3

flex-grow: 2;

Свободное пространство основной оси flex-контейнера будет распределено между flex-элементами пропорционально заданному значению flex-grow.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 1;

}

.flexElement:nth-of-type(2) {

flex-grow: 2;

}
