Деление свойств Flexbox

Свойства Flexbox делятся на свойства для родительского элемента (Flex-контейнера) и для дочерних элементов (Flex-элементов).

Свойства Flexbox для родительского элемента (Flex-контейнера)

  • display
  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

Свойства Flexbox для дочерних элементов (Flex-элементов)

  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • order
  • align-self
