flex-basis

Свойство устанавливает начальный основной размер flex-элемента до распределения свободного пространства основной оси в соответствии со значениями (%, em, rem, ...).

flex-basis: auto;

По умолчанию. Начальные размеры flex-элемента определяются заданным width и height, а также значением flex-grow.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 0;

flex-shrink: 1;

flex-basis: auto;

}

flex-basis: content;

Размеры Flex-элемента будут зависеть от содержимого. Не работает в Google Chrome.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 0;

flex-shrink: 1;

flex-basis: auto;

}

.flexElement:nth-of-type(1) {

flex-grow: 0;

flex-shrink: 1;

flex-basis: content;

flex-basis: значение;

Начальные размеры Flex-элементов будут определяться относительно Flex-контейнера. Но при заданном размере меньшем размеру контента, размер будет зависеть от width и height.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 0;

flex-shrink: 1;

flex-basis: auto;

}

.flexElement:nth-of-type(odd) {

flex-grow: 0;

flex-shrink: 1;

flex-basis: 40%;

}
