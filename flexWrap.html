flex-wrap

По умолчанию, все flex-элементы будут пытаться уместиться в одну строку. Свойство flex-wrap позволяет изменить это поведение и позволяет элементам размещаться в несколько строк при необходимости.

flex-wrap: nowrap;

Установлен по умолчанию. Flex-элементы будут пытаться разместиться в одну строку.

.flexContainer {

display: flex;

flex-direction: row;

flex-wrap: nowrap;

}
