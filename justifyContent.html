Flexbox Layout

Сайт создан в целях демонстрации модуля Flexbox Layout и обучения ему

justify-content

Определяет выравнивание вдоль основной оси (main axis). Это помогает распределить свободное пространство, оставшееся после того как все фиксированные и не фиксированные по ширине flex-элементы достигли максимального размера.

justify-content: flex-start;

Установлен по умолчанию. Flex-элементы будут прижиматься к основной оси.

.flexContainer {

display: flex;

flex-flow: row nowrap;

justify-content: flex-start;

}

justify-content: space-between;

Flex-элементы располагаются так, чтобы свободного пространства между первым элементом и началом основной оси, а также последнего элемента и концом основной оси, не было.

.flexContainer {

display: flex;

flex-flow: row nowrap;

justify-content: space-between;

}

justify-content: space-around;

Flex-элементы располагаются так, чтобы пространство возле них было одинаковым, но расстояние между элементами визуально не равно расстоянию между началом основной оси и первым элементом, а также последним элементом и концом основной оси, так как между flex-элементами будет двойное пространство, полученное суммой пространств flex-элементов.

.flexContainer {

display: flex;

flex-flow: row nowrap;

justify-content: space-around;

}

justify-content: space-evenly;

Похож на space-around, но расстояния между flex-элементами равны расстоянию между началом основной оси и первым элементом, а также последним элементом и концом основной оси.
