flex-shrink

Это свойство применяется к flex-элементу и определяет возможность элемента уменьшаться, при необходимости. Оно принимает безразмерное значение в качестве пропорции, которое определяет какое количество свободного пространства основной оси внутри контейнера должен занимать элемент. Нельзя указывать отрицательные значения.

flex-shrink: 0;

Flex-элементы не уменьшаются в размерах.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 0;

flex-shrink: 0;

}

Flex-элементы будут сужаться пропорционально заданному flex-shrink.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex-grow: 0;

flex-shrink: 1;

}

.flexElement:nth-of-type(even) {

flex-grow: 0;

flex-shrink: 2;

}
