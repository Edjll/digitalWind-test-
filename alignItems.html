Сайт создан в целях демонстрации модуля Flexbox Layout и обучения ему

align-items

Определяет выравнивание вдоль поперечной оси (cross axis). Похож на justify-content, но для поперечной оси.

align-items: stretch;

Установлен по умолчанию. Flex-элементы будут растягиваться на всю длину поперечной оси, но соблюдает размеры установленные элементу.

.flexContainer {

display: flex;

flex-flow: row nowrap;

justify-content: flex-start;

align-items: stretch;

}
