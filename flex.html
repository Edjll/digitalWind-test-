Flexbox Layout

Сайт создан в целях демонстрации модуля Flexbox Layout и обучения ему

flex

Сокращает запись flex-grow, flex-shrink, flex-basis. Задаются значения в виде flex: flex-grow flex-shrink flex-basis;. Flex-shrink и flex-basis указывать не обязательно.

flex: initial;

По умолчанию. Соответствует записи flex: 0 1 auto;.

.flexContainer {

display: flex;

flex-flow: row nowrap

justify-content: center;

align-items: center;

}

.flexElement {

flex: initial;

}
