Сайт создан в целях демонстрации модуля Flexbox Layout и обучения ему

align-content

Определяет выравнивание строк (или столбцов, зависит от направления основной оси) по поперечной оси (cross axis). Похож на align-items, но применяется ко всему содержимому строки (или столбца). Работает только при наличии нескольких строк (или столбцов)

align-content: stretch;

Установлен по умолчанию. Строки (или столбцы, зависит от направления основной оси) будут растягиваться по поперечной оси .Соблюдает размеры установленные элементу.

.flexContainer {

display: flex;

flex-flow: row nowrap;

justify-content: flex-start;

align-content: stretch;

}

align-content: space-around;

Строки (или столбцы) располагаются так, чтобы пространство возле них было одинаковым, но расстояние между строками (или столбцами) визуально не равно расстоянию между началом поперечной оси и первой строкой (или столбцом), а также последней строкой (или столбцом) и концом поперечной оси, так как между строками (или столбцами) будет двойное пространство, полученное суммой пространств строк (или столбцов).

.flexContainer {

display: flex;

flex-flow: row nowrap;

justify-content: flex-start;

align-content: space-around;

}
